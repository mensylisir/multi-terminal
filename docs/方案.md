# 一、产品目标

在浏览器中提供多主机实时终端能力，并支持安全可控的批量命令执行（Multiexec），同时具备审计与风险防护能力

---

# 二、核心能力

* 多主机 Terminal（每台独立）
* 多选主机 + 输入广播（Multiexec）
* 输出隔离 + 差异识别
* 会话恢复（断线重连）
* 审计（命令 + 回放）
* 风险控制（确认 / 限制 / 审批）
